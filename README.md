# docker-eyp_report

config variables:
```
DOC_URL_REST=""
DOC_USER=""
DOC_PASSWORD=""
DEBUG=0
DOC_ID=""
DOC_SPACE=""
MATRIX_ID=""
MATRIX_SPACE=""
```

* **DOC_URL_REST**: REST API URL
* **DOC_USER**: username
* **DOC_PASSWORD**: password
* **DEBUG**: enables/disables DEBUG valid values: 0/1
* **DOC_ID**: Page to publish list of modules
* **DOC_SPACE**: Space ID
* **MATRIX_ID**: Page to publish support matrix
* **MATRIX_SPACE**: Space ID
